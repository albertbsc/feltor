Detailed Description

template<class Geometry, class Matrix, class Vector>
struct dg::TensorElliptic< Geometry, Matrix, Vector >

Operator that acts as a 2d negative elliptic differential operator.

The term discretized is

\[ -\nabla \cdot ( \chi \cdot \nabla_\perp ) \]

where \(\chi\) is a symmetric tensor In general that means

\[ \begin{align} v^x = \chi^{xx} \partial_x f + \chi^{xy}\partial_y f \\ v^y = \chi^{yx} \partial_x f + \chi^{yy}\partial_y f \\ -\frac{1}{\sqrt{g}} \left(\partial_x(\sqrt{g} v^x ) + \partial_y(\sqrt{g} v^y) \right) \end{align} \]

Template Parameters
GeometryThe Geometry class to use
MatrixThe Matrix class to use
VectorThe Vector class to use
VectorThe Vector class to use This class has the SelfMadeMatrixTag so it can be used in blas2::symv functions and thus in a conjugate gradient solver.
Note
The constructors initialize \( \chi = I\)
Attention
Pay attention to the negative sign
