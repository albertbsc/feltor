Detailed Description

template<class FA, class Matrix, class container>
struct dg::DS< FA, Matrix, container >

Class for the evaluation of a parallel derivative.

This class discretizes the operators \( \nabla_\parallel = \mathbf{b}\cdot \nabla = b_R\partial_R + b_Z\partial_Z + b_\phi\partial_\phi \), \(\nabla_\parallel^\dagger\) and \(\Delta_\parallel=\nabla_\parallel^\dagger\cdot\nabla_\parallel\) in cylindrical coordinates

Template Parameters
FAEngine class for interpolation, provides the necessary interpolation operations
MatrixThe matrix class of the jump matrix
containerThe container-class on which the interpolation matrix operates on (does not need to be dg::HVec)
