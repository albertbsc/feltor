Detailed Description

A 3D grid class for cartesian coordinates.

In the third dimension only 1 polynomial coefficient is used, not n. In 2d it looks like

| -----> y
| |---x----------x---|
| |---x----------x---|
v |--- ---------- ---|
x |--- ---------- ---| fx*Lx
|--- ---------- ---|
fy*Ly
