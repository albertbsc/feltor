Detailed Description

High level matrix creation functions

Enumeration Type Documentation

enum dg::bc

Switch between boundary conditions.

Enumerator
PER 

periodic boundaries

DIR 

homogeneous dirichlet boundaries

DIR_NEU 

Dirichlet on left, Neumann on right boundary.

NEU_DIR 

Neumann on left, Dirichlet on right boundary.

NEU 

Neumann on both boundaries.

enum dg::direction

Direction of a discrete derivative.

Enumerator
forward 

forward derivative

backward 

backward derivative

centered 

centered derivative

enum dg::norm

Switch between normalisations.

Enumerator
normed 

indicates that output is properly normalized

not_normed 

indicates that normalisation weights (either T or V) are missing from output
