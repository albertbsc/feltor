Detailed Description

Class for 2d r2r transformation of Matrix using FFTW.

Actually just a wrapper for the r2r_2d plans and execute functions Note that RODFT00 IS significantly slower ( up to a factor of 10) than RODFT01. In or out-of place takes approx same time
