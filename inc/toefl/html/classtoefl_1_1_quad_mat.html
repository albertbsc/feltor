Detailed Description

template<class T, size_t n>
class toefl::QuadMat< T, n >

POD container for quadratic fixed size matrices.

The QuadMat stores its values inside the object. i.e. a QuadMat<double, 2> stores four double variables continously in memory. Therefore it is well suited for the use in the Matrix class (because memcpy and memset correctly work on this type)

Note
T and n should be of small size to reduce object size.
QuadMat is an aggregate so you can use initializer lists in c++11..
Template Parameters
Ttested with double and std::complex<double>
nsize of the Matrix, assumed to be small
